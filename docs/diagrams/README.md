# Diagramas del proyecto

Pega aca el PNG exportado desde `drawdb.app`.

Ruta sugerida:

- `docs/diagrams/db-diagram.png`

Export recomendado desde drawDB:

1. Abrir el diagrama.
2. Elegir `Export`.
3. Seleccionar `PNG`.
4. Guardarlo con el nombre `db-diagram.png`.
5. Copiarlo a esta carpeta.

Referencia rapida:

- El archivo puede verse desde [ARCHITECTURE.md](../ARCHITECTURE.md).
